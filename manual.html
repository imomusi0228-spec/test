VRC Cast Companion

VRChatイベント・店舗用 芳名帳＆キャストメモ連携システム 取扱説明書

1. System Overview (システム概要)

この度は「VRC Cast Companion」をご利用いただきありがとうございます。
本システムは、VRChatワールド内に設置する3D芳名帳と、キャストが手元（ブラウザ・スマホ）で確認する顧客管理Webダッシュボードをリアルタイムに同期・連携させるアセットパッケージです。

💡 特徴

・ゲストがワールドに入室した瞬間に自動検知。PC起動などの手間は一切なく、自動で来店回数が+1されます。
・入室と同時に、キャストのWeb画面が自動的にそのゲストのカルテページに切り替わります。
・ワールド内名簿には暗証番号（パスコード）ロック機能があり、一般ゲストからのデータ覗き見を防ぎます。

2. セットアップ手順

1 顧客管理ポータルでのアカウント作成と手帳ID（Book ID）の取得

1. 顧客管理ポータル（https://test-sepia-three-75.vercel.app）にアクセスし、新規アカウント登録（無料）を行います。
2. ログイン後、ダッシュボードの「手帳管理」画面より「新しい手帳」を作成します。
3. 作成された手帳の一覧に表示されている 手帳ID（Book ID） をコピーします。（このIDがVRChatワールドとデータベースを紐付けるキーになります）

2 Unity（VRChatワールド）への自動導入

1. 配布された 1_UnityAssets フォルダを、Unityプロジェクトの Assets フォルダ配下へドラッグ＆ドロップしてインポートします。
2. Unityの上部メニューバーに追加される VRC Cast Companion ➔ セットアップウィンドウ をクリックして開きます。
3. 開いたウィンドウに、以下の設定項目を入力します：

  • API Base URL : 初期設定のままで問題ありません（https://test-sepia-three-75.vercel.app/api）。
  • Book ID : Step 1でコピーした 手帳ID（Book ID） を入力します。
  • Passcode : ワールド内でキャスト専用の名簿閲覧画面を開くための任意の 暗証番号（4桁 of 数字など） を入力します。

4. 「自動セットアップを実行」 ボタンをクリックします。
これにより、シーン上のタブレット（Tablet）オブジェクトの上に美しくデザインされた「名簿UI画面(Canvas)」が自動設置され、スクリプトの変数アタッチやボタンイベントの紐付けがすべて自動で完了します。

3 ワールドのアップロード

1. キーボードの Ctrl + S を押して、シーンの変更を確実に保存します。
2. VRChat SDKのビルダー画面を開き、通常通りワールドのビルド＆アップロードを行ってください。
3. アップロード完了後、VRChat内で記名テストを行い、顧客管理ポータルのダッシュボードにリアルタイムで反映されれば接続完了です。

3. 動作仕様

自動来店カウント（12時間重複防止）

ゲストがワールドに入室した時点で、Udonが自動的にAPIを経由してデータベースへ入室通知を送ります。
この際、前回の入室日時から12時間以上経過していた場合、新しい「来店」とみなして来店カウントが自動的に `+1` されます。
12時間以内の再入室（回線落ちによる再接続など）のときは、来店数はカウントアップされずにログイン情報のみが記録されます。

一般ゲストとキャスト画面の保護

ゲスト用画面には個人情報（他プレイヤーの名前や来店回数の履歴）は一切表示されず、自身のコメント送信欄のみが提供されます。
事前に設定した暗証番号を入力してロックを解除することで、初めてキャスト用パネル（過去の記名履歴・来店回数一覧）が表示されます。これにより、ワールド内でのプライバシーが厳重に守られます。
