✨ 顧客魔導書 - クラウドDB初期設定

顧客魔導書へようこそ。
このアプリをクラウド（Supabase）と連携し、誰でもスマホ等から共同編集できるようにするため、以下の設定を行ってください。

🔑 設定手順
  1. Supabase (supabase.com) にサインアップし、新規プロジェクトを作成します。
  2. プロジェクト画面の Project Settings ➔ API ページを開き、Project URL と anon (public) キーをコピーし、以下に入力します。
  3. プロジェクト画面の SQL Editor ➔ New query を開き、マニュアルに記載されているSQLを貼り付けて Run ボタンを押します。

※ 接続に成功すると、必要なテーブルがデータベース内に自動構成されます。
※ 設定情報はブラウザとローカル設定ファイルに安全に保存されます。
